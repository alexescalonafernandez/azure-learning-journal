# Azure Learning Journal

This repository is my public learning journal for building practical capability in **Azure**, **Cloud**, **DevOps**, and **AI-assisted development**.

The goal is not to collect badges. The goal is to build real operational understanding through hands-on work, clear notes, and consistent execution over time.

---

## Purpose

This repo documents:

- What I am currently studying.
- What I have already practiced.
- What I have learned from mistakes and troubleshooting.
- What I plan to build next.

I use it as both a technical notebook and a professional evidence trail of my progress.

## Learning Philosophy

- **Learning by doing:** hands-on labs and small operational exercises before theory-only review.
- **Documenting progress:** write summaries, mistakes, and lessons learned after each learning block.
- **AI as copilot, not substitute:** use AI to accelerate feedback and drafting, while keeping technical judgment and verification as my responsibility.
- **Real skills over certificate collecting:** certifications are milestones, not the final objective.
