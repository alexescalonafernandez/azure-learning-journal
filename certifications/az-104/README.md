# AZ-104 Learning Hub

This folder is my practical learning hub for AZ-104.

The goal here is not to pass the exam by memorizing questions. The goal is to build real, operational understanding of Azure: how resources are organized, how to read the platform, and how to troubleshoot common issues with a clear method.

## Completed stages

### 1) Minimal operational foundations
I established the basic operational habits needed before touching Azure deeply: command-line confidence, structured notes, and a repeatable troubleshooting mindset.

### 2) Cloud foundations and Azure operational baseline
I worked through the core cloud concepts and Azure building blocks (subscriptions, resource groups, regions, and service models), with focus on how these ideas show up in real operations.

### 3) Core compute and storage in Azure
I practiced with virtual machines, app services, and storage accounts to understand the main compute/storage options and their operational trade-offs.

### 4) Operational networking and identity in Azure
I covered virtual networking and identity basics with an operator lens: connectivity paths, NSG-level reasoning, and fundamental access control behavior.

## Skills being built

- Azure resource model
- Azure Portal operational reading
- Basic identity and RBAC
- Virtual networking fundamentals
- Compute and storage fundamentals
- Troubleshooting by layers

## What is not assumed yet

- Advanced troubleshooting
- Advanced networking
- Advanced identity
- Automation/IaC mastery
- CI/CD
- Kubernetes
